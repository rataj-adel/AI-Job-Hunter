# AI Job Hunter Pipeline

## 📌 Overview
AI Job Hunter Pipeline is an AI-powered recruitment system that automates the hiring process by analyzing resumes, matching candidates with suitable job opportunities, and generating personalized interview questions using Natural Language Processing (NLP), Sentence Transformers (SBERT), and Large Language Models (LLMs).

---

## ✨ Features
- 📄 Resume (CV) text extraction from PDF files.
- 🧹 Text preprocessing and cleaning.
- 🛠️ Skill extraction and normalization.
- 🤝 Semantic job matching using SBERT embeddings.
- 📊 Candidate-job similarity scoring.
- 🎯 Ranking the most suitable job opportunities.
- 💬 AI-generated interview questions based on the candidate's skills and the selected job description.

## 🚀 How to Run


2. Install dependencies
